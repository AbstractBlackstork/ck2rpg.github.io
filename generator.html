Woohoo! You are done with the browser-based work. Here is what you need to do next:

  1. All of your files should be in the "replacers" folder in the template. If not, move them there.
  2. If you do not have Gimp for image processing, download it.
  3. Copy my Gimp python script out of this Github file.
  4. Open Gimp and navigate to Filters > Python-Fu > Console.
  5. In the console, locate the process_folder method and replace the placeholder path with the actual path to your replacers folder.
  6. Paste the edited script into the Python-Fu console and press Enter. This script applies a Gaussian blur, formats files, and places them in the correct folders.
  7. Open the "movetomapobjectdata" folder in the template. Copy all files from this folder and paste them into your "modname/gfx/map/map_object_data" to overwrite existing files.
  8. Open rivers.png in Gimp from the replacers folder. Open the existing rivers.png in the map_data folder. Copy and paste the generated rivers over the existing rivers.png in the map_data folder.
  9. Overwrite the template file in Gimp to ensure proper indexing.
  10. Open the papyrus.png file from the replacers folder in Gimp. Adjust brightness and apply a sepia filter if desired. Apply a Gaussian blur with a radius of 7.5. Open "flatmap.dds" in gfx/map/terrain/. Copy and paste your blurred map over the template flatmap.
  11. Open the map editor (right-click Crusader Kings 3 in Steam, click properties, and set launch options to -mapeditor).
  12. In the map editor, select Content Object Editor and mark all as modified.
  13. Repack the map and save it to your mod folder.
  14. Close the map editor.
  15. Delete all files from documents/paradox interactive/crusader kings 3/generated to gfx/map/map_object/data
  16. Change launch options to -debug_mode and start the game. Start as a ruler.
  17. Close the game to generate locators.
  18. Copy the generated locators from documents/paradox interactive/crusader kings 3/generated to gfx/map/map_object/data.
  19. You now should have a basic working map. Congratulations!
